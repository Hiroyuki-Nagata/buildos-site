OS自作入門メモ

１日目 - このサイトの構築 2016/03/05

はじめに

OS自作入門が何かということについてはググったほうが早いだろう。

WEBサーバ

とりあえず、成果を記録していくためにWEBサイトを用意することにした。はてなブログはこういうたぐいのサイトには便利だが、頻繁に更新するにはレスポンスが少し重い、 よって忍者ツールでサクッと作ることにした

デザイン

Bootstrapで作りたかったので、Honoka を使用している。これで日本語が美しく表示できる。

ソース管理・展開

忍者ホームページ はFTPでの管理となるため、若干ファイルの管理に不安がある。

GUIでやるのはだるいので、サイト内容はGitで管理し、アップロードはコマンドで行うことにした。 リポジトリはプライベートリポジトリが無料で作れるBitbucket を使い、そこからのアップロードは git-ftp を使う。完璧である。Windowsにコマンドがあるかわからないが、基本的にDebian であれば、apt-get install git-ftpで入ってくる。

展開手順

まずは秘密情報をGitのコンフィグに登録しておく。そしてリポジトリ自体プライベートなのでこれは関係者にしか見えないわけだ。

$ git config git-ftp.service.url ftp://domain.com/html/application/
$ git config git-ftp.service.user admin
$ git config git-ftp.service.password xxxxxxx

その後は一度git ftp initを打ち込めば、あとはソースをcommit&pushをしつつ変更点のみをサイトに更新していけるわけだ。戻したいと思った場合はGitの履歴をそこまで戻せば良い。

$ git ftp init -s service
$ git ftp push -s service

さあ、これで１日めは終了。
